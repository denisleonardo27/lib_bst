## BSTLIB

A library I developed to facilitate operations inside a BST data structure.
Every method is applied to the root node and the operations are:
- insertR(int k) -> recursive insertion of a new node where k is the value of the new node
- searchR(int k) -> recursive search of a value inside the tree
- inOrder() -> recursive way of printing the values in ascending order
- insertI(int k) -> iterative insertion of a new node where k is the value of the new node
- searchI(int k) -> iterative search of a value inside the tree
